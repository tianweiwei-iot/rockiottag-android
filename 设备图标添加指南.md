## 📦 图标规格要求

### 基本要求

| 属性 | 要求 | 说明 |
|------|------|------|
| **格式** | PNG | 支持透明背景 |
| **尺寸** | 80x80 px | 与播放标记大小一致 |
| **背景** | 透明 | 使用 Alpha 通道 |
| **颜色** | 鲜明 | 建议在地图上清晰可见 |
| **风格** | 简洁 | 图标线条清晰，不要太复杂 |

---

### 推荐规格（多分辨率适配）

为了在不同密度的屏幕上显示清晰，建议提供多个分辨率：

| 密度 | 目录 | 尺寸 |
|------|------|------|
| **mdpi** | `res/drawable-mdpi/` | 80x80 px |
| **hdpi** | `res/drawable-hdpi/` | 120x120 px |
| **xhdpi** | `res/drawable-xhdpi/` | 160x160 px |
| **xxhdpi** | `res/drawable-xxhdpi/` | 240x240 px |

**最少要求**: 至少提供 `drawable-xhdpi` (160x160 px) 版本

## 🎨 图标获取方式

### 方式 1: 使用免费图标网站（推荐）

#### Material Design Icons
- **网址**: https://fonts.google.com/icons
- **优点**: 风格统一，质量高
- **步骤**:
  1. 搜索关键词（如 "bicycle", "key", "pets"）
  2. 选择喜欢的图标
  3. 下载 PNG 格式（80x80 或 96x96）
  4. 重命名为 `ic_device_xxx.png`

#### Flaticon
- **网址**: https://www.flaticon.com/
- **优点**: 图标丰富，有彩色图标
- **步骤**:
  1. 搜索关键词
  2. 选择免费图标
  3. 下载 PNG 格式
  4. 调整大小为 80x80 px

#### Iconfont（阿里巴巴矢量图标库）
- **网址**: https://www.iconfont.cn/
- **优点**: 中文友好，图标丰富
- **步骤**:
  1. 搜索中文关键词（如"自行车"、"钥匙"）
  2. 选择图标
  3. 下载 PNG 格式
  4. 调整大小

---

### 方式 2: 使用 Android Studio 内置图标

1. 在 Android Studio 中右键点击 `res/drawable` 文件夹
2. 选择 **New** → **Vector Asset**
3. 选择 **Material Icon**
4. 搜索并选择图标
5. 设置名称为 `ic_device_xxx`
6. 点击 **Next** → **Finish**

**可用的 Material Icons**:
- 自行车: `directions_bike`
- 钥匙: `vpn_key`
- 宠物: `pets`
- 车辆: `directions_car`
- 背包: `backpack`
- 人员: `person`

## ⚠️ 常见问题

### Q1: 图标不显示怎么办？

**检查清单**:
1. ✅ 图标文件是否存在于 `res/drawable/` 目录
2. ✅ 文件名是否正确（全小写，下划线分隔）
3. ✅ 代码中的资源引用是否正确
4. ✅ 是否重新编译了项目

---

### Q2: 图标显示模糊怎么办？

**解决方案**:
1. 提供更高密度的图标（xhdpi、xxhdpi）
2. 确保图标尺寸至少 160x160 px
3. 使用矢量图标（Vector Drawable）

---

### Q3: 如何添加彩色图标？

**步骤**:
1. 准备彩色 PNG 图标
2. 确保背景透明
3. 复制到 `res/drawable/` 目录
4. 代码会自动使用彩色图标

---

### Q4: 想使用矢量图标怎么办？

**步骤**:
1. 在 Android Studio 中右键 `res/drawable`
2. 选择 **New** → **Vector Asset**
3. 选择 Material Icon
4. 命名并保存
5. 代码引用方式相同：`R.drawable.ic_device_xxx`
