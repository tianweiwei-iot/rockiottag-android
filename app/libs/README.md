# 高德地图SDK本地依赖说明

由于网络环境限制，无法从高德官方Maven仓库下载SDK文件，请按以下步骤手动准备依赖文件：

## 1. 下载高德地图SDK
访问高德地图开放平台（https://lbs.amap.com/）下载以下SDK的AAR/JAR文件：

- **3D地图 SDK**：版本 10.1.600
- **定位 SDK**：版本 6.5.1  
- **搜索 SDK**：版本 9.7.4

下载步骤：
1. 登录高德开放平台（账号：amapBllImsYDW）
2. 进入「控制台」→「应用管理」→「添加新应用」
3. 选择「Android平台」→「SDK下载」
4. 分别下载上述三个SDK的AAR格式文件

## 2. 文件命名规范
将下载的文件重命名为：
- `3dmap-10.1.600.aar`
- `location-6.5.1.aar`  
- `search-9.7.4.aar`

## 3. 放置文件
将重命名后的三个AAR文件复制到本目录（`app/libs/`）中。

## 4. 构建验证
完成文件放置后，在Android Studio中重新同步项目，Gradle将自动从本地libs目录加载依赖。

## 注意事项
- 确保文件版本与`app/build.gradle`中配置的版本号完全一致
- 如果下载的是JAR文件，需在`build.gradle`中相应调整依赖声明
- 本地依赖仅用于解决网络访问问题，不影响功能完整性
